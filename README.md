# ScollUI
## 介绍
### 学校进行的网页设计大赛项目，UI全部原生HTML5+CSS3设计的，逻辑部分用了部分Jquery框架，图片素材来自广大互联网，实现的功能主要有导航栏之高斯模糊，导航栏定位，适配器，动画等等。希望对您有一定的帮助。
